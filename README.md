# To-do list using Tailwind css+Animations and Mongodb
"TodoList crafted with Tailwind CSS for sleek styling and MongoDB for seamless data management. Enjoy smooth animations as you manage your tasks effortlessly. Stay organized, stay productive!"




## SETUP
Run the following commands to setup the tool :
```
git clone https://github.com/AshajanetR/Todo-list-Postgresql.git
```
```
cd client  npm i --y
```
```
cd server npm i--y
```
```
cd client npm start 
```
```
cd server npm start
```

## for mongodb compass connection
```
mongod
```
